# Courier Logos

This directory stores logo images for shipping couriers.
The system will automatically look for images in this folder matching the courier code.

## Supported File Names

- `jne.png`, `jne.jpg`, `jne.webp`
- `jnt.png`
- `sicepat.png`
- `pos.png`
- `tiki.png`
- `lion.png`
- `ninja.png`
- `anteraja.png`
- `idexpress.png`
- `wahana.png`

## Instructions

1. Upload your logo image here.
2. Rename it to match the courier code (lowercase).
3. The checkout page will automatically display it.
